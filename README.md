This repository presents a hybrid deep learning framework for breast cancer detection using a combination of EfficientNetV2 and Change Vision Transformer (Change ViT). The proposed approach integrates convolutional and transformer-based architectures to effectively capture both local spatial features and global contextual dependencies from medical images. A cross-attention based feature fusion mechanism is employed to enhance interaction between heterogeneous feature representations, followed by HBPSO-AL optimization to select the most discriminative feature subset. Additionally, advanced preprocessing, data augmentation, and AG-StyleGAN-based synthetic image generation are incorporated to improve data quality and model generalization. The framework also includes Grad-CAM visualization to provide interpretability by highlighting clinically relevant regions. Experimental results demonstrate high accuracy, robustness, and strong generalization capability, making the proposed system suitable for reliable computer-aided breast cancer diagnosis.
